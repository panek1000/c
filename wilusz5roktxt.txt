This GPS fix worked in my F5. It comes from DOOGEE support.

Enter Engineering menu (Mobileuncle Tools or *#*#3646633#*#)

Select Engineering mode for MTK 

Select YGPS -> Information -> and tap on 'AGPS restart' (press once and you will see nothing)

Exit YGPS and select LocationBasedService. 

Select AGPS -> Enable A-GPS 

Select GPS 

Select FULL 

Tap on Ref Position and see if some location datas are present: enter your current position (Lat,Long) if it's empty. Select ok.

Let the GPS start by tapping on GPS (OFF). It will become 'ON' and will procede with fixing the position

Tap on GPS (ON), it will become OFF (Leave it ON at least for three minutes then OFF)

Select HOT instead of FULL

Enter Ref Position and tap 'Apply Last Position'. (It will apply your current position)

Reboot

Install GPS Status & Toolbox and see if it fixes satellites easily 

I hope it works for you
